Power BI Sales Analysis Project

This project is created to analyze sales data using Power BI. The main
goal is to understand business performance and apply DAX functions.

Data Tables Used: - Sales_Fact - Customer_Dim - Product_Dim -
Region_Dim - Date_Dim - Returns_Fact

Calculated Columns: - Profit = SalesAmount - Cost - ReturnFlag to
identify returned and not returned items - Customer Full Name by
combining first and last name

Measures Created: - Total Sales - Total Cost - Total Profit - Return
Rate (%) - Average Sale per Transaction

Quick Measures: - Year-over-Year (YoY) Sales Growth - Difference between
current and previous month sales

Measure Management: - Created a dedicated Measures table to organize all
measures

Filter Context Analysis: - Used ALL(), FILTER(), and CALCULATE() to
compare sales with and without filters

DAX Functions Used: - Math: SUM, AVERAGE, MAX - Count: COUNTX,
DISTINCTCOUNT - Logical: IF, AND, OR, SWITCH - Text: CONCATENATE, UPPER,
LEFT - Date: YEAR, MONTH, EOMONTH

Relationships: - Connected Sales_Fact with all dimension tables - Used
RELATED() to bring data like Customer Name, Product Name, and Region

Time Intelligence: - Used TOTALYTD, SAMEPERIODLASTYEAR, DATESINPERIOD -
Created Running Total using CALCULATE and DATESBETWEEN

Additional Scenarios: - Categorized sales into Low, Medium, High using
SWITCH - Used SUMX and AVERAGEX for advanced calculations

Conclusion: This project demonstrates how to use Power BI and DAX to
analyze sales data, create reports, and gain business insights.
